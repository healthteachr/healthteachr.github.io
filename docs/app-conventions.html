Appendix D — Conventions

This book will use the following conventions:

  • Generic code: list(number = 1, letter = "A")
  • Highlighted code: list(number = 1, letter = “A”)
  • File paths: data/sales.csv
  • Menu/interface options: Tools > Global Options… > Pane Layout
  • R Packages: tidyverse
  • Functions: paste()
  • Strings: “Lisa”, “Emily”
  • Numbers: 100, 3.14
  • Logical values: TRUE, FALSE
  • Glossary items: ordinal
  • Citations: Wickham (2021)
  • Internal links: Chapter 3
  • External links: Mastering Shiny
  • Mac-specific: Cmd-Shift-F10
  • Windows-specific: Ctl-Shift-F10
